# XMRig Miner

Find and edit the 'config.json', with minimum config as follows:

    "http": {
        "enabled": true,
        "host": "0.0.0.0",
        "port": 8989,
        "access-token": null,
        "restricted": true
    },

Port number above can be whatever you want it to be, but it must then match what is specified in the WTH config.
